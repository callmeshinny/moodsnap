# MoodSnap

Minimal private photo-sharing prototype with mood-color posting.

## Features
- Upload a photo, toggle **Moody Mode**, pick a mood color, add optional caption, and share.
- Friends feed shows mood-colored photo frames.
- Mood filter tab to browse posts by mood.
- Weekly mood colour summary counts mood posts from last 7 days.
- In-app push-style notifications and simulated real-time friend posts.

## Run locally
This repository currently has no build/test tooling. You can run a simple static server, for example:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.
