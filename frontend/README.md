# MoodSnap Frontend

MoodSnap mobile app built with Expo Router and React Native.

## Get Started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

The main route files live in `src/app`.

## Scripts

- `npm start` - start Expo through tunnel.
- `npm run start:lan` - start Expo on LAN.
- `npm run ios` / `npm run android` / `npm run web` - platform-specific starts.
